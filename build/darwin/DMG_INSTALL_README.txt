Gobrew Installation Guide (macOS)

1. Drag "Gobrew.app" to the "Applications" folder shortcut.
2. Open Launchpad or /Applications, then start Gobrew.
3. If macOS blocks launch on first run:
   - Right-click Gobrew.app and choose "Open", then confirm.

---

Gobrew 安装说明（macOS）

1. 将 "Gobrew.app" 拖拽到 "Applications" 文件夹快捷方式。
2. 在 Launchpad 或 /Applications 中启动 Gobrew。
3. 若首次启动被系统拦截：
   - 右键 Gobrew.app，选择“打开”，再确认。
